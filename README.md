# Winbakk Solution

Landing page desenvolvida para apresentar a proposta da Winbakk, uma solucao SaaS focada em recuperar clientes inativos e transformar relacionamento em receita recorrente.

## Visao geral

O projeto comunica o posicionamento da plataforma de forma objetiva, com foco em conversao, clareza da proposta de valor e apresentacao visual moderna. A pagina foi estruturada para destacar beneficios, processo de uso, resultados esperados, FAQ e apresentacao do time.

## Stack

- HTML
- CSS
- JavaScript

## Destaques

- Hero com proposta de valor, metricas e CTAs de conversao
- Secao explicando o funcionamento da plataforma em 3 passos
- Bloco de resultados com indicadores de ROI e engajamento
- FAQ com acordeao interativo
- Secao institucional com equipe e contatos
- Visual dark com efeito glassmorphism e foco em apresentacao premium
- Estrutura leve e facil de publicar

## Estrutura do projeto

```text
assets/
index.html
styles.css
styles-tech.css
script.js
design-guidelines.json
winbakk-solution-context-profile.md
```

## Como executar

- Abra `index.html` no navegador
- Ou use uma extensao como `Live Server` no VS Code/Cursor

## Arquivos importantes

- `index.html`: landing page principal
- `script.js`: interacoes basicas e comportamentos de interface
- `design-guidelines.json`: referencia visual do sistema de design
- `winbakk-solution-context-profile.md`: documento com contexto estrategico do produto

## Objetivo do projeto

Demonstrar a criacao de uma landing page para produto digital com foco em posicionamento, conversao, experiencia visual e comunicacao clara de valor.
